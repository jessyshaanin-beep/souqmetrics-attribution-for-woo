Capture marketing attribution data (UTMs and click IDs) and attach it to WooCommerce orders.

== Description ==

SouqMetrics – WooCommerce Attribution is a lightweight plugin that captures marketing attribution data and stores it directly on WooCommerce orders.

The plugin works automatically after activation and requires no configuration.

It is designed to support analytics and reporting tools by making attribution data available via WooCommerce order metadata.

== Features ==

* Captures UTM parameters (utm_source, utm_medium, utm_campaign, utm_content, utm_term)
* Captures click IDs (fbclid, gclid, ttclid)
* Stores attribution data on WooCommerce orders
* Compatible with Cash on Delivery and paid orders
* No configuration required
* No external API calls

== Installation ==

1. Go to Plugins → Add New
2. Search for "SouqMetrics WooCommerce Attribution"
3. Click Install and Activate

== Frequently Asked Questions ==

= Does this plugin send data externally? =
No. All data is stored locally on the WooCommerce order.

= Does this plugin track users? =
No. It only captures marketing parameters present in the URL at the time of visit.

= Is WooCommerce required? =
Yes. This plugin only works with WooCommerce installed.
